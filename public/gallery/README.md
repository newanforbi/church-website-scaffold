# Gallery Images

Drop image files here and reference them in `lib/gallery-data.ts` (matching the `src` path).

The gallery currently expects these five files, matching the flyers already described in `lib/gallery-data.ts`:

- `welcome-to-worship.jpg`
- `digging-deep.jpg`
- `new-month-power.jpg`
- `power-night.jpg`
- `shiloh-hour.jpg`

Tips:

- JPG or WebP, roughly 1200px on the long edge, keeps page weight down.
- Use descriptive filenames — they don't need to match exactly, just update `src` in `lib/gallery-data.ts` to match.
- To add more images, append new entries to the `galleryImages` array in `lib/gallery-data.ts`.
